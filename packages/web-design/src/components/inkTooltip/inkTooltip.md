# InkTooltip

A tooltip component that shows content on hover.

## Props

- `content`: The tooltip content text
- `position`: Position of the tooltip ('top' | 'bottom' | 'left' | 'right')

## Slots

- Default slot: The trigger element

## Usage

```vue
<InkTooltip content="This is a tooltip" position="top">
  <button>Hover me</button>
</InkTooltip>
```
